# Git/GitHub Cheatsheet

## Create

#### Create a new local repo

```txt
git init
```

#### Create a new local repo in a specific directory

```txt
$ git init <directory>
```

#### Clone existing repo

- HTTPS

```txt
$ git clone https://github.com/user/repo.git
```

- SSH

```txt
$ git clone ssh://user@domain.com/repo.git
```
